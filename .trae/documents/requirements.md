# 大众点评网站应用需求文档

## 项目概述
开发一个简化版的大众点评网站应用，提供用户注册登录、商家信息展示、用户评价和搜索功能。

## 功能需求

### 1. 用户系统
- **用户注册/登录**: 支持手机号+验证码或邮箱+密码方式
- **个人资料管理**: 头像、昵称、联系方式等基本信息管理
- **用户权限管理**: 区分普通用户和商家用户权限

### 2. 商家模块
- **商家信息展示**: 名称、地址、联系方式、营业时间等基本信息
- **商家分类检索**: 餐饮、娱乐、购物等分类浏览
- **地理位置展示**: 集成地图API显示商家位置

### 3. 点评模块
- **用户评价功能**: 星级评价(1-5星) + 文字评价
- **评价展示**: 按时间排序，支持分页显示
- **商家回复功能**: 商家可以回复用户评价

### 4. 搜索功能
- **基础搜索**: 按商家名称、分类、地理位置搜索
- **筛选功能**: 按评分、距离等条件筛选

## 技术需求
- **前端**: React + TypeScript + Tailwind CSS，响应式设计
- **后端**: Node.js + Express + TypeScript，RESTful API
- **数据库**: PostgreSQL (通过Supabase)
- **部署**: 云服务器部署
- **地图服务**: 集成地图API

## 非功能性需求
- 响应式设计，适配PC和移动端
- 良好的用户体验和界面设计
- 基本的性能优化
- 安全性考虑（用户认证、数据验证等）

## 交付标准
1. 可运行的完整系统原型
2. 基础功能测试报告
3. 简单的使用说明文档
4. 源代码和部署指南
